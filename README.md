# Golang API boilerplate
![Golang](https://blog.skillfactory.ru/wp-content/uploads/2021/12/image2-8611380.png)
### Includes:
  - Gin
  - Goloby IoC container
  - Postgres driver
  - JWT token authorization
  - API versioning
  - Cors middleware
  - Module structure
  - Built-in features like guards, filters etc.
